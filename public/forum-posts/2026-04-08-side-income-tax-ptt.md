## 標題

【報稅攻略】上班族有副業，5月報稅前必看：副業30萬只多繳8,250元稅的原因

---

## 內文（完整版，適合 Dcard 理財版 / 工作板）

各位好，

快到報稅季了，身邊有不少人有副業（接案、外送、教學、出租...），但每次遇到5月就緊張，不確定要怎麼報稅、會多繳多少。

整理一下幾個重點，讓大家有個底。

---

### 副業所得分四種，申報方式不同

很多人搞混的是：**接案收入 ≠ 薪資所得**，不能直接當薪資申報。

| 所得類型 | 適用情況 | 費用扣除率 |
|---------|---------|---------|
| 執行業務所得（9A） | 接案設計/撰稿/程式/顧問/教學 | **30%~62%**（依職業類別） |
| 薪資所得（50） | 打工兼差/外送（多數平台） | 薪資扣除額（全年上限 NT$207,000） |
| 租賃所得 | 出租房間/車位/Airbnb | 必要費用率43% |
| 其他所得 | 偶發網拍/版稅/競賽獎金 | 扣必要成本 |

→ 看你的扣繳憑單，「所得類別」欄位是「50」還是「9A」，決定用哪種方式。

---

### 重點：執行業務費用率

接案族最有利的制度：政府認定你的收入有一定比例是「成本」，不課稅。

常見費用率（114年度）：
- 廣告設計、美術設計：**45%**
- 撰稿、程式設計、講師：**30%**
- 會計師、律師：**30%**
- 醫師、牙醫：**62%**（最高）

→ 接案廣告設計年收 30 萬，課稅所得只有 **30 萬 × 55% = 16.5 萬**

---

### 三種族群實際試算（114年度）

**案例 A：上班族（薪資50萬）+ 接案設計（30萬）**

| 項目 | 計算 | 金額 |
|---|---|---|
| 薪資所得淨額 | 50萬 - 薪資扣除20.7萬 | 29.3萬 |
| 執行業務所得 | 30萬 × (1-45%) | 16.5萬 |
| 免稅額（單身） | | -9.2萬 |
| 標準扣除額（單身） | | -13.1萬 |
| 綜合所得淨額 | | 23.5萬 |
| 適用稅率 | 5% | |
| 應繳稅額 | | **NT$11,750** |
| 純薪資50萬時稅額 | | NT$3,500 |
| **副業30萬多繳** | | **NT$8,250** |

副業多賺 16.5 萬淨所得，只多繳 **NT$8,250**，有效稅率不到 3%。

---

**案例 B：上班族（薪資50萬）+ 出租房間（月租15,000元）**

| 項目 | 計算 | 金額 |
|---|---|---|
| 年租收入 | 15,000 × 12 | 18萬 |
| 租賃所得 | 18萬 × (1-43%) | 10.26萬 |
| 薪資所得淨額 | | 29.3萬 |
| 合計淨額（扣免稅額/扣除額） | | 17.26萬 |
| 應繳稅額 | 5% | **NT$8,630** |
| 純薪資時稅額 | | NT$3,500 |
| **出租多繳** | | **NT$5,130** |

月收租 15,000，每年只多繳 **NT$5,130** 稅。

---

**案例 C：外送員注意**

外送收入（UberEats/Foodpanda 薪資憑單）通常是「薪資所得（代碼50）」，沒有費用率優惠。

薪資扣除額一年只有 NT$207,000，主職薪資已用完的情況下，兼職外送每一塊錢都全額計入課稅。

→ 正職月薪4萬，外送月收 10,000 元（年12萬）：兼差部分幾乎全額課稅，稅率5%，年多繳約 **NT$6,000**

---

### 副業不申報有風險

很多人「收入不多，懶得報」，但：

1. 扣繳憑單已自動送給稅局，漏報系統會比對
2. 漏報被查到：補稅 + **罰40%罰款**，比稅本身更貴
3. 執行業務有費用率，你可能反而**少繳稅**，不申報反而虧

## 版本 B（PTT 精簡版，適合 Salary板）

**標題：【報稅攻略】有副業上班族：接案30萬只多繳8,250元稅，為什麼？**

快到5月，有副業的上班族注意一件事：

**接案收入不是薪資，有「費用率」可以扣。**

以接案廣告設計為例（費用率45%）：
- 接案30萬 → 課稅所得只算 30萬×55% = **16.5萬**
- 加回正職薪資50萬，算完只多繳 **NT$8,250**
- 副業多賺16.5萬淨所得，有效稅率不到3%

各職業費用率一覽：
- 廣告/美術設計：45%
- 程式/撰稿/講師：30%
- 醫師：62%（最高）

→ 先確認你的扣繳憑單代碼是「9A」（執行業務）還是「50」（薪資）

算你自己的情況 → https://twtaxcalc.com/side-income-tax

注意：漏報扣繳憑單有補稅+40%罰款，不要賭。
