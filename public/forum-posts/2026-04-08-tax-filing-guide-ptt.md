### 114年度關鍵數字速查

**稅率級距（課稅所得）：**

| 課稅所得 | 稅率 |
|---|---|
| 0 ~ 590,000 | 5% |
| 590,001 ~ 1,330,000 | 12% |
| 1,330,001 ~ 2,660,000 | 20% |
| 2,660,001 ~ 4,980,000 | 30% |
| 4,980,001 以上 | 40% |

**免稅額：**
- 本人 / 配偶 / 扶養親屬：各 NT$97,000
- 70歲以上（本人、配偶、扶養父母）：NT$145,500

**扣除額（擇標準或列舉，取較高）：**
- 標準扣除額：單身 NT$131,000 / 夫妻合併 NT$262,000
- 列舉扣除額：房貸利息 + 捐贈 + 保險費 + 醫藥費 + 房租（各有上限）

**特別扣除額（符合條件直接扣）：**

| 項目 | 金額 |
|---|---|
| 薪資特別扣除額 | NT$218,000/人（上限） |
| 幼兒學前扣除（第1個未滿6歲） | NT$120,000 |
| 幼兒學前扣除（第2個以上） | NT$240,000 |
| 教育學費（大學以上就讀） | NT$25,000/人 |
| 身心障礙特別扣除 | NT$218,000/人 |
| 長期照顧特別扣除 | NT$120,000/人 |

### 各類所得怎麼申報

**薪資族：**
- 雇主已預扣稅款，報稅主要是確認扣除額填齊
- 重點：薪資特別扣除額（NT$218,000）、配偶或父母是否列扶養

**有股利的投資人：**
- 選一：股利合併所得稅（稅率依你的課稅所得級距）+ 8.5% 抵減（最高NT$80,000）
- 選二：28% 分離課稅（固定28%，但無法抵減）
- 年薪較低、稅率 5~12% 的人通常合併課稅比較省

**租金收入：**
- 租金屬於「租賃所得」，需申報
- 可選必要費用（43%費率）或核實列報（房屋折舊＋實際費用）

**副業/接案收入：**
- 屬於「執行業務所得」，依所得類別有費用率扣除（一般服務業50%）
- 淨所得才計入課稅所得

**海外所得：**
- 超過 NT$1,000,000 且基本所得稅 > NT$670,000 才可能需要繳AMT
- 美股ETF持有但未售出：通常不需額外申報

---

### 四個常見漏掉的扣除額

1. **幼兒學前特別扣除**：有未滿6歲幼兒的，每個小孩可額外扣NT$120,000（很多人忘填）
2. **房貸利息**：列舉申報時，可扣除已繳利息（上限NT$300,000），有房貸的人通常值得試算
3. **身心障礙/長期照顧**：家中有符合資格者，每人可額外扣NT$120,000~218,000
4. **受扶養大學生教育學費**：就讀大學以上每人可扣NT$25,000（需有繳費收據）

### 網路申報流程（5步驟）

1. 前往財政部報稅網站或下載報稅軟體
2. 用自然人憑證或健保IC卡登入
3. 系統自動帶入薪資、股利等資料，確認是否正確
4. 補填扶養親屬、各項扣除額
5. 試算後確認送出，完成申報

網路申報可直接試算退補稅，建議在5月底前儘早送出（退稅批次越早越快）。
