**標題：剛處理完父親遺產，整理遺產稅申報眉角分享（附免費計算工具）**

---

事情是這樣的。

三個月前父親因病辭世，身為長子負責處理遺產稅申報。以前完全沒概念，以為就是「遺產 × 稅率」，後來花了兩個週末研究才搞清楚，整理出來給也可能需要的人。

---

**最大的誤解：遺產稅不是用市價算**

遺產裡有房子，計入遺產的不是「市場成交價」：
- 土地：用**公告現值**（通常是市價的 40–60%）
- 房屋：用**評定現值/課稅現值**（通常是市價的 10–20%）

舉例：台北市大安區一間市值 2,500 萬的老公寓
→ 土地公告現值可能 400 萬
→ 房屋評定現值可能 200 萬
→ 計入遺產只有 600 萬，不是 2,500 萬

這就是為什麼很多人的遺產稅遠比想像中低。

---

**現行扣除額（114 年度）**

| 項目 | 金額 |
|------|------|
| 免稅額 | **1,333 萬** |
| 喪葬費 | 最高 138.3 萬 |
| 配偶 | 553 萬 |
| 每位成年子女 | 55.3 萬 |
| 每位未成年子女 | 55.3 萬 + 每差1歲加55.3萬 |
| 父母（每人） | 123.3 萬 |
| 重度以上障礙繼承人 | 693 萬/人 |
| 受扶養兄弟姊妹/祖父母 | 55.3 萬/人 |

「標準四口之家」（有配偶 + 2 成年子女）扣除額：
1,333 + 138.3 + 553 + 55.3 × 2 = **2,134.9 萬**

遺產不到 2,134 萬 → 課稅淨額 0 → 遺產稅 0

---

**稅率**

| 課稅遺產淨額 | 稅率 |
|------------|------|
| 0 ~ 5,000 萬 | 10% |
| 5,000 萬 ~ 1 億 | 15% |
| 超過 1 億 | 20% |

---

**重要！不用繳稅也要申報**

這個很多人踩坑。

就算計算出來不用繳稅，還是要在**死亡後 6 個月內**（可申請延期 3 個月）向所在地稽徵機關申報，取得：
- 「遺產稅繳清證明書」（需繳稅者）
- 「遺產稅免稅同意書」（免稅者）

**沒有這兩份文件 → 無法辦理房地產繼承過戶**

我當初不知道這件事，差點到了地政事務所才發現沒辦法過戶。

最近幫家人處理遺產事宜，整理了一下遺產稅計算邏輯，分享給有需要的人。

台灣遺產稅（114 年度）重點：

1. 免稅額 1,333 萬（先扣這個）
2. 喪葬費最高 138.3 萬
3. 配偶 553 萬、成年子女每人 55.3 萬、未成年子女額外加（每差一歲多 55.3 萬）
4. 不動產用公告現值 + 評定現值，非市價
