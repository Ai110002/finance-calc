## PTT 版本（Salary板 / home-sale板）

**標題：** [請益] 出租房屋租賃所得怎麼報稅？費用率43%實際差多少

**內文：**

各位板友好，

最近繼承了一間老公寓，打算出租，月租大概 $20,000-$25,000，
查了一下要申報租賃所得，想整理一下重點分享給跟我一樣不熟悉的板友。

---

**一、租賃所得怎麼計算？**

台灣租賃所得有兩種計算方式：

**方法一：費用率43%**（最常用、最簡單）
- 租賃所得 = 年租金 × (1 - 43%) = 年租金 × 57%
- 政府假設你有 43% 是合理成本（折舊、修繕、管理費等）
- 不需要收集任何憑證，直接用

**方法二：列舉實際費用**
- 租賃所得 = 年租金 - 實際費用（需憑證）
- 實際費用包含：維修費、管理費、建物折舊、房屋稅、地價稅、貸款利息
- 適合實際費用超過43%的房東

---

**二、實際稅負試算（114年度）**

月租 $20,000，全年出租，薪資年收 $60 萬的上班族：

| 項目 | 金額 |
|------|------|
| 年租金 | $240,000 |
| 費用扣除（43%） | $103,200 |
| 租賃所得 | $136,800 |
| 薪資所得 | $600,000 |
| 薪資特別扣除 | $218,000 |
| 標準扣除額 | $131,000 |
| 免稅額 | $97,000 |
| 綜合所得淨額 | $290,800 |
| 適用稅率 | 5% |
| 應納稅額 | $14,540 |

如果沒有出租，薪資只有 $60 萬：
- 綜合所得淨額：$154,000
- 應納稅額：$7,700

因此出租後多繳稅：$6,840 / 年，月均 $570

對月租 $20,000 來說，稅後實拿約 $19,430 / 月。

---

**三、月租超過 $20,000 的注意事項**

如果月租金單次超過 NT$20,000，承租方（租客或公司）需要代扣**二代健保補充保費 2.11%**。

月租 $25,000 的話：
- 補充保費 = $25,000 × 2.11% = $527.5 / 月
- 年補充保費約 $6,330

這筆錢是從你的租金扣掉的，實際拿到會少一點。

---

**四、什麼情況用列舉實際費用比較好？**

如果你有：
- 大額維修費（換管線、翻新廁所）
- 高額貸款利息
- 房屋稅 + 地價稅金額偏高

實際費用加總超過年租金的 43% → 用列舉比較划算。

舉例：月租 $30,000（年租 $36 萬），實際費用（修繕$8萬 + 貸款利息$7萬 + 稅費$2萬）= $17 萬 > 43% ($154,800)

用列舉：租賃所得 = $360,000 - $170,000 = $190,000
用費用率：租賃所得 = $360,000 × 57% = $205,200

→ 這種情況列舉省更多稅。

## Dcard 版本（理財版）

**標題：** 出租房子要繳多少稅？費用率43%怎麼算，附完整試算

**內文：**

最近很多人問我出租房屋的稅怎麼算，整理一下分享給大家。

**核心概念：政府承認43%是成本**

不管你有沒有收據，政府都認定你的租金收入有 43% 是維修、折舊等成本，所以只有 57% 要算所得稅。

這叫「費用率43%法」，是大多數房東用的方式。

**計算公式：**
租賃所得 = 年租金 × 57%

**實際案例**（上班族兼出租）：

月薪 $4 萬（年薪 $48 萬）+ 月租 $20,000：

- 租賃所得 = $240,000 × 57% = $136,800
- 合計所得 = $480,000 + $136,800 = $616,800
- 扣除標準扣除 $131,000 + 薪資扣除 $218,000 + 免稅額 $97,000
- 應稅所得 = $170,800（適用5%）
- 全年稅 = $8,540

如果沒出租：應稅所得 $34,000，稅 $1,700

→ 多繳稅 $6,840 / 年，出租月收 $20,000，實際多繳 $570 / 月

**月租超過 $20,000 的房東**要注意，租客（或公司）每次付租金時會代扣 2.11% 的二代健保補充保費，實際拿到的租金會少一點。

**什麼時候不用費用率，改用列舉？**
→ 有大額裝修、高額貸款利息、實際費用超過年租金 43% 時
