## Dcard 理財版長文版

**標題：114年度「房屋租金特別扣除額」完整攻略 — 租屋族可以少繳多少稅？**

今年（2026年5月）報稅有個很多上班族不知道的新制，在這裡整理給大家。

### 什麼是「房屋租金特別扣除額」？

114年度起，租屋族可以額外申報房租支出作為扣除額，**每申報戶上限 NT$180,000**。

最重要的差異在：這是「**特別扣除額**」，不是「列舉扣除額」。

- **以前（113年度以前）**：租金只能用「列舉扣除」申報，要放棄標準扣除額（131,000元）才行
- **現在（114年度起）**：可以同時享有「標準扣除額」＋「房屋租金特別扣除額」，兩個疊加

### 有什麼條件？

1. **本人及配偶在台灣境內無自有房屋**（最關鍵，含配偶名下）
2. 租賃供本人、配偶或受扶養親屬實際居住使用
3. 不可向直系親屬（父母、子女）租賃
4. 需有合法租賃契約及付款紀錄

### 能省多少稅？

以月薪 60,000（年薪 720,000）、月租 15,000、單身申報舉例：

| 項目 | 金額 |
|------|------|
| 年薪所得 | 720,000 |
| 薪資特別扣除額 | −218,000 |
| 個人免稅額 | −97,000 |
| 標準扣除額 | −131,000 |
| **房屋租金特別扣除額** | **−180,000** |
| **申報後所得淨額** | **94,000** |
| 應繳稅額（5%） | 4,700 |
| 不申報租金扣除時應繳稅 | **13,900** |
| **省稅金額** | **NT$9,200** |

薪資更高（12%稅率）的人省更多：同樣月租 15,000，一年可省約 **NT$21,600**。

### 要怎麼申報？

5月報稅時，在財政部綜合所得稅申報系統填寫「房屋租金特別扣除額」欄位，輸入全年租金金額（系統自動套用上限18萬）。

需要保留文件：租賃契約 + 付款紀錄（轉帳明細 or 收據）。
